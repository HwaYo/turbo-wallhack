How-To-Use
==========
```
$ mkdir dir_name
$ cd dir_name
$ git remote add https://github.com/HwaYo/turbo-wallhack.git
$ git pull
```
These commands will setup the codes in your computer
```
$ bundle install
$ ruby scraper.rb
```
This will fire up a browser(Firefox at default) and wallhack will start crawling!
